# CS 4347 - Team 8 - UTD Link

## Getting Started

Execute the following commands in a terminal to set up the project:

- `git clone \<url>`
- `cd utd-link`
- `python -m venv .venv`
- `.venv\Scripts\activate.bat` (for Windows) <br /> `source .venv/bin/activate` (for Mac/Linux)
- `pip install -r requirements.txt`
- Copy the contents of `.env.example` file into `.env` file with the correct secrets

## Running the program

`python app.py`
